STEPS TO RUN !!
1. pip install uv (https://docs.astral.sh/uv/getting-started/installation/)
2. uv venv
3. source .venv/bin/activate    # on mac
   .venv\Scripts\activate       # on windows
4. uv add -r requirements.txt
5. uv run python -m vectorstore.py
6. uv run python app.py
